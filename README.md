# mobile-jwt-login


1. npm install

2. npx sequelize init // 안해도 됨

3. fill out config/config.json  // 내 user id, password 설정

ex)

"development": {
    "username": "root",
    "password": "password",
    "database": "login",
    "host": "127.0.0.1",
    "dialect": "mysql"
  },

4. npx sequelize db:create

5. npm start
